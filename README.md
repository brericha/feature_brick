To use this brick you need to have `flutter_bloc` and `equatable` in your pubspec.yaml

To add this brick

```
$ mason init (if you have not already)
$ mason add feature --git-url https://github.com/brericha/feature_brick
```

To add a feature

From project root:
```
$ cd lib
$ mason make feature
```
